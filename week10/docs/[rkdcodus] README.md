# 스크롤 사이트 제작하기

- 네비게이션 바의 메뉴 요소를 클릭하면 해당 섹션으로 화면이 자동 스크롤된다.
- 초기 화면에 정가운데 있는 버튼을 클릭하면 가장 마지막 섹션으로 이동한다.
- 반응형 네비게이션 바
  - 브라우저 너비가 50% 이하가 되면 네비게이션 바의 메뉴 아이콘을 클릭하여 메뉴 요소를 확인할 수 있다.
  - 메뉴 요소를 클릭하면 해당 섹션으로 이동하며 열려있던 메뉴바는 닫힌다.
